## 🎯 סקירה כללית

### חזון המוצר
**FlashLearn** היא פלטפורמת למידה אינטראקטיבית המשלבת כרטיסי פלאש מונפשים עם בינה מלאכותית ליצירת חוויית למידה מותאמת אישית, מהנה ואפקטיבית.

### הבעיה שאנו פותרים

| בעיה | השפעה |
|------|-------|
| שיטות למידה מסורתיות משעממות | 67% מהתלמידים מאבדים עניין |
| קושי בשינון חומר יבש | שכחה של 80% מהחומר תוך שבוע |
| חוסר התאמה אישית | כל תלמיד לומד בקצב שונה |
| חוסר משוב מיידי | תלמידים לא יודעים היכן לשפר |
| תוכן לא מעודכן | ספרי לימוד מיושנים |

### הפתרון שלנו
פלטפורמה שמשלבת:
- 🎴 **כרטיסי פלאש מונפשים** - ויזואליים ומעניינים
- 🤖 **AI מותאם אישית** - יוצר תוכן ומתאים קושי
- 🎮 **גיימיפיקציה** - נקודות, הישגים, תחרויות
- 📊 **מעקב התקדמות** - למידה מבוססת נתונים
- 👥 **למידה חברתית** - שיתוף ותחרות בריאה

### 2. 🤖 מנוע AI

#### יכולות

```javascript
// יצירת כרטיסים אוטומטית
AI.generateCards({
  topic: "מלחמת העולם השנייה",
  level: "כיתה י'",
  count: 20,
  types: ["text", "visual", "interactive"]
});

// התאמת קושי
AI.adaptDifficulty({
  userId: "12345",
  performance: {
    correct: 15,
    incorrect: 5,
    averageTime: "4.2s"
  }
});

// יצירת הסברים
AI.explainAnswer({
  question: "מהי פוטוסינתזה?",
  wrongAnswer: "שריפת סוכר",
  correctAnswer: "יצירת סוכר מאור שמש",
  userLevel: "beginner"
});

// שאלות המשך
AI.generateFollowUp({
  previousQuestion: "...",
  wasCorrect: false,
  focusArea: "הבנה עמוקה"
});
```

### מבנה מסד נתונים

```sql
-- משתמשים
CREATE TABLE users (
  id UUID PRIMARY KEY,
  email VARCHAR(255) UNIQUE,
  name VARCHAR(100),
  role ENUM('student', 'teacher', 'admin'),
  level INT DEFAULT 1,
  xp INT DEFAULT 0,
  streak_days INT DEFAULT 0,
  created_at TIMESTAMP
);

-- חפיסות
CREATE TABLE decks (
  id UUID PRIMARY KEY,
  creator_id UUID REFERENCES users(id),
  title VARCHAR(200),
  description TEXT,
  subject VARCHAR(100),
  grade_level VARCHAR(50),
  is_public BOOLEAN DEFAULT false,
  card_count INT,
  created_at TIMESTAMP
);

-- כרטיסים
CREATE TABLE cards (
  id UUID PRIMARY KEY,
  deck_id UUID REFERENCES decks(id),
  card_type ENUM('text', 'visual', 'audio', 'video', 'interactive'),
  front_content JSONB,
  back_content JSONB,
  animation VARCHAR(50),
  difficulty INT DEFAULT 1,
  created_at TIMESTAMP
);

-- התקדמות למידה
CREATE TABLE learning_progress (
  id UUID PRIMARY KEY,
  user_id UUID REFERENCES users(id),
  card_id UUID REFERENCES cards(id),
  ease_factor FLOAT DEFAULT 2.5,
  interval_days INT DEFAULT 1,
  repetitions INT DEFAULT 0,
  next_review TIMESTAMP,
  last_review TIMESTAMP
);

-- היסטוריית תשובות
CREATE TABLE answer_history (
  id UUID PRIMARY KEY,
  user_id UUID REFERENCES users(id),
  card_id UUID REFERENCES cards(id),
  was_correct BOOLEAN,
  response_time_ms INT,
  answered_at TIMESTAMP
);
```

## 🧠 אלגוריתם למידה

### Spaced Repetition System (SRS)

מבוסס על אלגוריתם SM-2 עם התאמות:

```python
def calculate_next_review(card, quality):
    """
    quality: 0-5 (0=שכח לגמרי, 5=זכר בקלות)
    """
    if quality < 3:  # תשובה שגויה
        card.repetitions = 0
        card.interval = 1
    else:
        if card.repetitions == 0:
            card.interval = 1
        elif card.repetitions == 1:
            card.interval = 6
        else:
            card.interval = round(card.interval * card.ease_factor)

        card.repetitions += 1

    # עדכון ease factor
    card.ease_factor = max(1.3,
        card.ease_factor + (0.1 - (5 - quality) * (0.08 + (5 - quality) * 0.02))
    )

    card.next_review = today + card.interval
    return card
```

### התאמות ייחודיות

```
1. גיל המשתמש
   ילדים (6-12): intervals קצרים יותר
   מבוגרים: intervals סטנדרטיים

2. סוג התוכן
   מילים: intervals ארוכים
   נוסחאות: intervals קצרים

3. שעה ביום
   בוקר: קושי גבוה יותר
   ערב: קושי נמוך יותר

4. היסטוריה אישית
   שגיאות חוזרות: יותר חזרות
   הצלחה עקבית: פחות חזרות
```
